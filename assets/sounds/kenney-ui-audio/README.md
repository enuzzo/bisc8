# Kenney UI Audio candidates

These WAV files are copied from Kenney UI Audio.

Source: https://kenney.nl/assets/ui-audio
License: Creative Commons CC0 1.0 Universal

The current firmware uses a generated lightweight chime instead of embedding one
of these WAV files. The ten files in this folder are short CC0 candidates to
audition on hardware later.

Provisional favorite: `switch12.wav`.
