Browser serial flasher

Bisc8

Install the public firmware, then use the Bisc8 hotspot to configure Wi-Fi, language, OpenAI, and email.

Flash

Use Chrome, Edge, or another Web Serial browser. Open this page over HTTPS or localhost.

Public firmware contains no OpenAI keys, email relay tokens, Wi-Fi credentials, or private SMTP/provider secrets.

After Flash

  1. Wait for the board to restart.
  2. Join the Bisc8-XXXX Wi-Fi network.
  3. Open http://192.168.4.1 if the captive page does not appear.
  4. Save Wi-Fi, language, OpenAI key, and optional email recipient.

Recovery

  1. Hold BOOT during startup to force setup mode.
  2. Hold BOOT + PWR to clear local configuration.
  3. Use serial commands WIFI SETUP, WIFI RESET, CONFIG RESET, or STATUS.
Secrets entered later are stored on the device. Production units should enable flash encryption.
